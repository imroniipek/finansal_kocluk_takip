# Finansal Koçluk Takip Uygulaması

Kişisel finans yönetimini kolaylaştırmak için geliştirilmiş kapsamlı bir **Flutter** uygulaması.  
Kullanıcılar gelir/gider takibini yapabilir, aylık/yıllık harcama istatistiklerini görebilir ve  
BLoC mimarisi ile çalışan dinamik bir arayüz sayesinde gelişmiş bir deneyim elde eder.

---

## Özellikler

- **Gelir / Gider Yönetimi**
  - Yeni gelir ve gider ekleme
  - Harcama kategorisi seçimi
  - Not ekleyebilme
- **BLoC Mimarisi ile State Management**
  - Temiz, test edilebilir ve scalable yapı
  - Event & State tabanlı kontrol
- **SQLite Lokal Veri Tabanı**
  - Veri kaybı olmadan offline kullanım
  - Tablolar:
    - `income`
    - `expenses`
- **Tarih Seçimi & Formatlama**
- **Sayısal Giriş Mantığı (Hesap makinesi tarzında)**
- **Modern ve Responsive Flutter UI**
- **Çoklu Platform Desteği**
  - Android
  - iOS
  - Web
  - Windows
  - MacOS
